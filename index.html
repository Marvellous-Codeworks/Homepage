A new house for TMS

It’s the add-on for Google Chrome, it turns browser into what it has never been in it’s life: a browser that is no longer resource killer (and if you have ever used Google’s browser you know very well what I’m talking about, with the results on Windows or macOS). The Marvellous Suspender has everything you need to allow you to control the behavior of active / inactive tabs and thus free up RAM when possible, in a transparent way, you do the first configuration and then you forget it exists, what you can do is only that couple of clicks that will serve to manually suspend a tab that you have awakened by mistake by selecting it.

Fork of the original The Great Suspender, but without those modifications that go to invade user privacy. You can find the official repository at github.com/gioxx/MarvellousSuspender, feel free to participate if you have the skills to do that.
